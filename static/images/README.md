## Chat Background Image (chat-background.jpg)

### Background Image Setup

1. Save your desired image as `chat-background.jpg` in this folder.
2. Recommended format: JPG, PNG
3. Recommended size: 1920x1080 or larger
4. Image will be automatically scaled and centered.

### Current Settings

- Background image is applied to the entire chat area
- 70% opacity white overlay ensures text readability
- Subtle pattern overlay adds visual depth
- `backdrop-filter: blur(8px)` creates a soft blurred background effect

### Image Guidelines

- Avoid overly complex or dark images that may reduce text readability
- Use bright, simple images for best results
- Compress image files appropriately to avoid long loading times
